# b07 bot

## Getting Started

1. Install Python and Twisted
2. Clone the source:

   git clone https://github.com/dirtydave0221/b07bot.git
   cd b07

3. Copy the sample config file and edit it:

   cp b07.sample ~/.b07
   edit ~/.b07

4. Run the server:

   PYTHONPATH=src python src/b07/main.py
